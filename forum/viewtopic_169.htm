by Magic » Thu Jul 05, 2012 1:13 am

you post a c# keygen - well I presuppose you know its almost the same in c# ?!


the easiest way:
    1. OpenProcess
    2. get MainModule.BaseAddress
    3. read ReadUInt( with BaseAddress)
    4. add your 1st Offset to the ReadUInt value
    5. read ReadUInt( with CurrentAddress )
    6. add your 2st Offset to the ReadUInt value
    7. read ReadUInt( with CurrentAddress )
    8. - n. ...
    n. after all offsets read/write to your last CurrentValue
done.

the short way C#
Code: Select all
WriteProcessMemory(processHandle, (void*) (address + offset), &value, sizeof(value), &bytesread)


the short way C++
Code: Select all
WriteProcessMemory / ReadProcessMemory(processHandle, (LPVOID) address, &value, sizeof(int), NULL);

more specifically if you only patch

you want
005F4283 - 8B 55 F4 - MOV EDX,DWORD PTR SS:[EBP-C]
patch to
005F4283 - 8B D6 - MOV EDX,ESI


just OpenProcess -> create byte arrays -> patch

c++
Code: Select all
handle = OpenProcess (...) // OpenProcess
      BYTE ADD[3] = {0x8B,0xD6}; // create byte array  8B D6
      BYTE NOP[1] = {0x90}; // *
      WriteProcessMemory(handle, (void*)0x005F4283,ADD, 3, new DWORD) // patch @ 005F4283
      WriteProcessMemory(handle, (void*)0x005F4285,NOP, 1, new DWORD)// *

you have 3 bytes at 005F4283 - 8B 55 F4 and want to patch the 8B 55 F4 -> 8B D6 ofc you have to nop F4 -> NOP
