Code: Select all
00B11404  |> \C60424 00      MOV BYTE PTR [ESP],0


to

Code: Select all
00B11404  |> \C60424 01      MOV BYTE PTR [ESP],1


1 byte change.

Note: Because crackme is linked with ASLR enabled, addresses will be different. Can use this pattern to find the patch:

Code: Select all
8A 54 04 04 3A 90 ?? ?? ?? ?? 75 08 40 83 F8 1F 7C EE EB 04 C6 04 24 00

Re: Second Challenge: Patchme1 by R4ndom

by random » Tue Aug 28, 2012 3:55 pm

Memory relocation is used if the Windows loader attempts to load an executable at a memory location that is already used. ASLR is used to force this memory relocation every time, regardless of if there is a program loaded in the desired memory location or not.

Generally, an exe file will asked to be loaded at 401000, and most often, will be loaded at that address. ASLR tells the loader "please pick a random memory location to load me at and change all of the references in the exe to reflect this".
